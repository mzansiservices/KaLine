How It Works

Get started in just 3 simple steps

1

Add Dependency

Add our package to your pubspec.yaml file

dependencies:
  notification_system: ^1.0.0
2

Initialize in Your App

Set up the notification service in your main.dart

await NotificationService().initialize(
  appId: 'your_app_id',
  appName: 'Your App Name'
);
3

Send Notifications

Start sending notifications with one line of code

NotificationService().send(
  title: 'Hello',
  body: 'Welcome to our app!',
  data: {'key': 'value'}
);
